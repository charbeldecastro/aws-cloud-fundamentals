# Cloud Foundations

Este módulo apresenta os fundamentos essenciais da Computação em Nuvem, abordando conceitos, modelos de serviço e arquitetura básica das soluções cloud.

---

## Objetivo

Compreender:

- O que é Computação em Nuvem
- Características fundamentais do modelo cloud
- Provedores de nuvem
- Modelos de implantação
- Modelos de serviço (IaaS, PaaS, SaaS)
- Arquitetura de soluções em nuvem
- Benefícios estratégicos e técnicos

---

## Estrutura do Módulo

| Capítulo | Tema |
|----------|------|
| 01 | Conceitos Fundamentais |
| 02 | Modelos de Serviço |
| 03 | Arquitetura e Benefícios |

---

## Resultado Esperado

Ao final deste módulo, o estudante deve ser capaz de:

- Explicar o funcionamento da computação em nuvem
- Diferenciar modelos de implantação
- Comparar IaaS, PaaS e SaaS
- Identificar componentes arquiteturais de uma solução cloud
