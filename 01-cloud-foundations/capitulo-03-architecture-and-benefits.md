# Capítulo 03 – Arquitetura e Benefícios da Nuvem

## Estrutura Arquitetural Básica

Soluções em nuvem são compostas por três camadas principais:

1. Front-end
2. Back-end
3. Gestão e Segurança

---

## Front-end

Ponto de interação com o usuário.

Exemplos:
- Navegadores
- Aplicativos móveis
- Interfaces web

Funções:
- Receber entradas
- Exibir dados
- Comunicar-se via APIs

---

## Back-end

Responsável pelo processamento e regras de negócio.

Componentes:
- Servidores virtuais
- Bancos de dados
- APIs
- Microserviços

Funções:
- Processar requisições
- Garantir integridade dos dados
- Executar rotinas críticas

---

## Armazenamento

Serviços responsáveis por guardar dados estruturados e não estruturados.

Exemplos:
- Amazon S3
- Azure Blob Storage

---

## Rede e Conectividade

Interligação entre componentes.

Tecnologias:
- Load Balancers
- Gateways de API
- VPN
- SDN

---

## Gerenciamento e Automação

Ferramentas que reduzem intervenção manual:

- Monitoramento
- Auto Scaling
- Provisionamento via scripts
- CI/CD

---

## Segurança

Mecanismos:

- IAM
- Criptografia
- Firewalls virtuais
- MFA
- Políticas de permissão

---

## Benefícios da Computação em Nuvem

- Elasticidade
- Redução de custos
- Alta disponibilidade
- Agilidade
- Escala global
- Segurança escalável

---

## Tendências

- Serverless
- Edge Computing
- AI como serviço
- Data centers sustentáveis
