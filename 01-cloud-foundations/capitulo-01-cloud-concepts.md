# Capítulo 01 – Conceitos Fundamentais de Computação em Nuvem

## Definição

Computação em Nuvem é um modelo de entrega de recursos de tecnologia da informação sob demanda, via internet, com pagamento baseado no consumo.

Permite acesso remoto a:

- Servidores
- Armazenamento
- Redes
- Bancos de dados
- Aplicações

---

## Características Fundamentais

### 1. Sob Demanda
Provisionamento automático sem necessidade de intervenção humana direta.

### 2. Acesso via Internet
Disponível a partir de qualquer dispositivo conectado.

### 3. Pay-as-you-go
Pagamento baseado no uso real dos recursos.

### 4. Escalabilidade e Elasticidade
Ajuste dinâmico de capacidade conforme necessidade.

### 5. Multilocação
Compartilhamento seguro da mesma infraestrutura entre múltiplos clientes.

---

## Provedores de Nuvem

Empresas responsáveis pela infraestrutura:

- AWS
- Microsoft Azure
- Google Cloud Platform

---

## Modelos de Implantação

### Nuvem Pública
Infraestrutura gerenciada por provedores terceirizados.

### Nuvem Privada
Ambiente exclusivo para uma organização.

### Nuvem Híbrida
Combinação entre ambientes públicos e privados.
